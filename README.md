# Injeção de dependências
Descrição do projeto: Um código que aborda maneiras de injetar dependencias.
## O que é a Injeção de depêndencia?
-> Padrão de projeto para evitar o alto nível de acoplamento do código.
### O que isso influência?
-> Melhora a legibilidade do código, interpretação, manutenabilidade e distribuição de responsabilidade de classes(inversão de controle). 
## Injeção de dependência e Inversão de controle
-> Injeção de dependência é uma das duas maneiras de implementar inversão de controle.
## O que é Inversão de Controle?
-> Se refere a troca de responsabilidade que passa a ser do consumidor da classe e não da própria classe

## O que você deverá esperar desse projeto?
-> Irei deixar o código de injeções de projeto atráves de:
- Construtor
- Interface
- Propriedade
